# GST Franchise Center

Modern React + TypeScript + Tailwind CSS rebuild of the [GST Franchise Center](https://github.com/Karan227/gst_website) website.

## Stack

- **React 19** + **TypeScript** (`src/index.tsx`)
- **Vite** — dev server and production builds
- **Tailwind CSS v4** — styling
- Original brand images preserved in `public/img/`

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

```bash
npm run build
npm run preview
```

## Live Demo (GitHub Pages)

**https://vivekcyr25.github.io/gst_website/**

Deployed automatically from `main` via GitHub Actions.

## Features

- Responsive header with smooth-scroll navigation
- Hero carousel with high-resolution slides
- Primary services grid with modern icons
- FAQ accordion, testimonials, service listings
- Contact footer with working section links
